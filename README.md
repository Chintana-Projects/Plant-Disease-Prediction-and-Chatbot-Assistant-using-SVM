# 🌿 Plant Disease Prediction and Chatbot Assistant using SVM

## 📖 Overview

This project is a Flask-based web application that helps users identify plant diseases from leaf images and provides an AI-powered chatbot for agricultural assistance. The system uses a Support Vector Machine (SVM) model for disease prediction and an interactive chatbot to answer farming-related queries.

---

## ✨ Features

* 🌱 Upload plant leaf images for disease prediction
* 🤖 AI-powered chatbot for agriculture-related questions
* 📊 Machine Learning model using Support Vector Machine (SVM)
* 🖥️ User-friendly Flask web interface
* 📂 Image upload and prediction history support
* 📱 Responsive web design

## 📸 How It Works

1. Upload a plant leaf image.
2. The image is preprocessed.
3. The trained SVM model predicts the disease.
4. The prediction result is displayed.
5. Users can also interact with the chatbot for farming guidance and disease information.
